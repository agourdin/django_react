REFER TO: https://wsvincent.com/django-rest-framework-react-tutorial/


Get up and running:
1. `cd proj/backend/` and run `pipenv shell` then run `python manage.py runserver`
2. Go to http://127.0.0.1:8000/admin/ to play around with the model example
   in admin mode.
3. Go to http://127.0.0.1:8000/app/ to see the rest_framework API console and
   check out the JSON output.
4. `cd proj/frontend/` and run `npm start`
5. Go to http://localhost:3000/ to see the app
6. Edit `proj/frontend/src/App.js` to change the app!
7. Edit `proj/frontend/public/index.html` to customize the look!
